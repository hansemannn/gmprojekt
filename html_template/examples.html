Ist das Fenster kleiner als 767px werden die Spalten "fluid", das heißt sie werden untereinander angeordnet und nehmen automatisch die volle breite ein.

Grid HTML

Ein Grid lässt sich mit einer kleinen anzahl an CSS Klassen sehr leicht realisieren. Für ein einfaches, zweispaltiges Layout reicht es, eine .row Element anzulegene und die passende anzahl an .span* Elemente darin zu platzieren. Da es sich um ein 12-Spaltiges Grid handelt, wird jedes .span* den entsprechenden anteil dieser 12 Spalten einnehmen. 

<div class="row">
  <div class="span5">...</div>
  <div class="span2">...</div>
  <div class="span5">...</div>
</div>

In diesem Beispiel haben wir zwei .span5 und ein .span2, welche zusammen 12 Spalten ergebene und so eine komplette Zeile im Grid-System belegen.

Thumbnails

Bildergallerie

Das Grid-System biete auch eine tolle und einfache möglichkeit Bilder, Videos, Produktinfos, Portfolios und vieles mehr ansprechend auf der Seite darzustellen.

HTML Markup

Wie bereits erwähnt ist der benötigte HTML Code wieder sehr kompakt und einfach. Dies ist ein Auszug zur Erstellung der Thumbnail Ansicht wie so oben zu sehen ist. 

<ul class="thumbnails">
  <li class="span3">
    <div class="thumbnail">
      <img src="images/thumbnail1.jpg">
    </div>
  </li>
  ...
</ul>
